# CODSOFT Machine Learning Internship

This repository contains the projects completed during my Machine Learning internship at CODSOFT. These projects demonstrate the application of various ML algorithms to solve real-world classification and prediction problems.

## Projects Completed:

### 1. Movie Genre Classification 
* **Goal:** Predict the genre of a movie based on its plot summary.
* **Algorithm:** Logistic Regression / Naive Bayes.
* **Tech:** Python, Pandas, Scikit-learn.

### 2. Customer Churn Prediction 
* **Goal:** Analyze customer data (Complaints/Care) to predict if a user will leave the service.
* **Algorithm:** Random Forest / Logistic Regression.
* **Tech:** Data Cleaning, Feature Engineering, Classification.

### 3. Spam SMS Detection 
* **Goal:** Build a model to classify messages as "Spam" or "Ham" (Real).
* **Algorithm:** Multinomial Naive Bayes.
* **Tech:** NLP (Natural Language Processing), CountVectorizer, Accuracy Evaluation.
